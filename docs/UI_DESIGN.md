# UI Design Specification

## Design System

### Theme
- **Primary framework:** DaisyUI
- **Preferred theme:** `dark` or `night` (configurable)
- **Mobile-first:** Design for 375px+ width first
- **Language:** Dutch (Nederlands)

### Colors (DaisyUI semantic colors)
```css
primary    - Main actions (Join, Create)
secondary  - Secondary actions
accent     - Highlights
neutral    - Text and backgrounds
success    - Positive states (completed, points gained)
warning    - Caution states
error      - Errors, negative points
info       - Information
```
